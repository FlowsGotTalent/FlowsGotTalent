# Flows Got Talent

# Front End 

## Project Setup
### Main requirement:
Vue3 Vueitfy3, Materio Theme

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

#Backend
Firebase

## Project Setup
`npm install -g firebase-tools`
`firebase int`
## Deploy
`firebase deploy`
